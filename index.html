Payment Fraud Detection System

Machine Learning-Powered Transaction Analysis

This project demonstrates a comprehensive ML pipeline for detecting fraudulent online payment transactions. Built on a dataset of over 6 million records, the system employs 8 distinct classification algorithms to identify suspicious patterns.

Dataset

  • Source: Kaggle — Online Payment Fraud Detection
  • Size: 6,362,620 transaction records
  • Fraud cases: 8,213 (0.13%)
  • Features: step, type, amount, balances, isFraud
  • Preprocessing: Downsampling, feature engineering, StandardScaler

Tech Stack

Python scikit-learn pandas NumPy Matplotlib Seaborn Jupyter StandardScaler

Models Implemented

  • Logistic Regression
  • Naive Bayes (GaussianNB)
  • Decision Tree Classifier
  • Random Forest Classifier
  • Gradient Boosting Classifier
  • Perceptron
  • K-Nearest Neighbors (KNN)
  • Support Vector Machine (SVC)
